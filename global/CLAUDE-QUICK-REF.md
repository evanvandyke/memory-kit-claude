# [USER_NAME]'s Quick Reference

You don't need to memorize any of this. It's here when you need to look something up.

## Session Skills

These were installed by the Memory Kit. You can call them directly or just say what you need in plain language.

| Skill | What to say | When |
|---|---|---|
| `/start` | "let's get started" | Beginning of every session |
| `/wrap` | "wrap it up" | End of every session |
| `/agenda` | "add this to my agenda" or "what's on my agenda?" | Anytime. Also runs automatically. |
| `/compress` | "save my progress" | Mid-session if things run long |
| `/project-setup` | "set up this project" | Once per new project |
| `/project-repair` | Runs automatically inside `/wrap` | Every 5 sessions. Hands-free. |

## Claude Code Controls

These are built into Claude Code. Type them directly in the conversation (except the first one, which you type when launching Claude from the terminal).

| Command | What it does | When |
|---|---|---|
| `claude --dangerously-skip-permissions` | Start Claude without approval prompts | When launching from the terminal |
| `/model` | Switch between AI models | Beginning of a session |
| `/effort` | Set how deeply Claude thinks | Beginning of a session |
| `/context` | See how much of the conversation window is used | Mid-session |
| `/usage` | See session cost, rate limit, and quota | Anytime |

## The Daily Rhythm

1. Open your project in VS Code
2. Open the terminal (View > Terminal)
3. Type `claude` and hit enter
4. Say "let's get started"
5. Work normally. Talk about what you're doing.
6. Say "wrap it up" when you're done
